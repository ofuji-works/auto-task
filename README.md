# 自動登録ツール

## 初期設定

**1. node のインストール**

```
https://nodejs.org/ja/download/
```

コマンドラインツールを開く(windows ならコマンドプロンプト等)

以下コマンドで node、npm がインストールされていることを確認

```
$ node --version
$ npm --version
```

**2. ツールの準備**

コマンドラインでツールのディレクトリ内に入り、package.json があることを確認。
以下コマンドを実行する。

```
$ npm install
```
